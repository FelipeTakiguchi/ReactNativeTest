
# React Native Pratice




## Instalação

Instale com npm

```bash
  npm install
  cd ReactNativeTest
```

Crie um Projeto React Native

```bash
    npm install --global yarn
    yarn add expo
    npx create-expo-app MeuProjeto
    cd MeuProjeto
    npx expo install react-native-web@~0.18.10 react-dom@18.2.0 @expo/webpack-config@^18.0.1
```
